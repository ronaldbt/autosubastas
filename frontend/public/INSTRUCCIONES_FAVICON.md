# Instrucciones para crear el Favicon PNG/ICO

El favicon usa el mismo icono **Car de Lucide** que aparece en el sidebar.

## Opción 1: Usar el generador HTML (Recomendado)

1. Abre el archivo `public/generate-favicon-png.html` en tu navegador
2. Haz clic en el botón "Descargar PNG (32x32)"
3. Guarda el archivo como `favicon.png` en la carpeta `public/`
4. (Opcional) Para crear el `.ico`, visita https://favicon.io/favicon-converter/ y convierte el PNG a ICO

## Opción 2: Usar herramienta online

1. Visita https://favicon.io/favicon-converter/
2. Sube el archivo `public/favicon.svg`
3. Descarga el `favicon.ico` generado
4. Reemplaza `public/favicon.ico` con el archivo descargado

## Opción 3: Usar RealFaviconGenerator

1. Visita https://realfavicongenerator.net/
2. Sube el archivo `public/favicon.svg`
3. Configura los tamaños y descarga el paquete
4. Reemplaza los archivos en `public/`

El icono Car de Lucide ya está configurado en el SVG y se verá igual que en el sidebar.
