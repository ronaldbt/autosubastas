# Favicon Setup

El favicon de la aplicación está configurado como SVG (`favicon.svg`) que funciona en navegadores modernos.

## Para generar un favicon.ico (opcional, para compatibilidad)

Si necesitas generar un `favicon.ico` desde el SVG, puedes usar:

1. **Herramientas online:**
   - https://favicon.io/favicon-converter/
   - https://realfavicongenerator.net/
   - https://convertio.co/svg-ico/

2. **Pasos:**
   - Sube el archivo `favicon.svg`
   - Descarga el `favicon.ico` generado
   - Reemplaza el archivo `public/favicon.ico`

El SVG ya está configurado y funcionará en la mayoría de navegadores modernos.
